## Quick start

Add PrismMCP to your MCP client config:

```json
{
  "mcpServers": {
    "prismmcp": {
      "command": "path/to/PrismMCP-Shim",
      "args": ["--port", "55557"]
    }
  }
}
```

Works with Claude Code, Cursor, Claude Desktop, and any MCP-compatible agent.
