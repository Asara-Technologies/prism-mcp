# PrismMCP Extension Examples

These examples are documentation fixtures. They are intentionally stored under `docs/` so they do not auto-register in
an editor session.

- `cpp/MyInventoryExtension.cpp` shows direct C++ command registration.
- `blueprint/README.md` outlines a Blueprint Toolset DataAsset setup.
- `python/inventory/manifest.json` and `python/inventory/handlers.py` show the Python pack layout.
